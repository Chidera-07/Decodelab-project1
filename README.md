
# Decodelab-project1
Data cleaning and transformation project using Excel and Power Query to improve data quality, handle missing values, remove inconsistencies, and prepare data for analysis. [Dataset for Data Analytics2.xlsx](https://github.com/user-attachments/files/28470211/Dataset.for.Data.Analytics2.xlsx)


## Data Cleaning and Transformation

This project focused on preparing raw transactional data for analysis using Microsoft Excel and Power Query. The dataset was assessed for data quality issues, including missing values, inconsistencies, and formatting errors.

Using Power Query, I cleaned and transformed the data by standardizing formats, validating records, and handling missing values. Null entries in the **Coupon Code** column were replaced with **"NO COUPON"** to accurately represent transactions where no discount code was applied. This ensured data completeness and improved the reliability of subsequent analysis.

In addition, the date column was standardized by converting it into a proper date format using Power Query. This ensured consistency across all records and enabled accurate time-based analysis such as trend evaluation and performance tracking.

These transformations improved the overall data quality and made the dataset data analysis and visualization.

### Tools Used

* Microsoft Excel
* Power Query

### Key Tasks Performed

* Identified and handled missing values
* Replaced null values in the Coupon Code field with "NO COUPON
" using replace value in Power Query
* Standardized date formats
* Validated data consistency
* Prepared data for analysis and visualization
